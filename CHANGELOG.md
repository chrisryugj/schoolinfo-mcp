## [0.3.0] - 2026-06-20

**포지셔닝 개편 — "매일 일상보조" → "학교 비교·수행평가·내신 인텔리전스"**. 외부 피드백
(현직 교사 3 + 초등 학부모 1) 기반. 일상정보(급식·일정)는 incumbent(e알리미 등)와 겹쳐
격하하고, incumbent가 못 채우는 빈틈(학교 비교·hwp 속 수행평가)을 전면으로. **기능 삭제 0.**
계획: `docs/2026-06-19-repositioning-school-intelligence.md`.

### Changed
- **카피·메타 전면 교체**: title/og/twitter/hero/eyebrow/README/package를 급식 우선 →
  학교 비교·수행평가 우선으로. 기능 변경 없이 검색·공유 노출 메시지 전환.
- **웹앱 모드탭 2 → 3개**: `📊 학교 비교`(신규·기본 진입) / `📋 수행평가·내신` / `🎓 대학 진학`.
  학교 비교는 지역+학교급만으로 비교표 첫 화면(학교를 먼저 안 찾아도 됨).
- **급식·이번주·학사일정**을 학교카드 전면 → 접이식(`🍱 …더보기`)으로 격하.

### Added
- **학년별·학급별 학생수(공시 09) 구조화 표** — 기존 `| 항목 | 값 |` 나열로 학년이 섞여
  못 읽겠다는 학부모 피드백. 학년을 열로 정리 + 학급수·학급당·교사 1인당. 학년 컬럼이 없는
  양식은 generic 표로 폴백. (MCP `get_disclosure`/digest 공통)
- **평가계획 교과 바로가기 칩** — 통합형 평가계획은 전 교과가 한 표에 이어져 교과를 못 찾는
  문제. "과목/교과" 머리글이 있는 표의 교과명 셀로 바로가기 칩 생성(표 구조 불변, 없으면 무영향).

### Notes
- 하위호환 유지: MCP 도구 시그니처·반환 타입 불변(마크다운 내용만 가독성 개선).

## [0.2.0] - 2026-06-16

학원·학부모를 위한 **다(多)학교** 기능 2종 추가. 단일 학교 조회를 넘어 지역 단위로 시험·지표를 비교한다.

### Added
- **지역 시험 캘린더** — 여러 학교의 중간·기말고사 일정을 한 타임라인으로 집계.
  - MCP `get_exam_calendar`, CLI `schoolinfo exams`, 웹앱 학교카드 `📝 시험 캘린더`.
  - 입력 하이브리드: `names`(특정 학교 배열) 또는 `sgg`+`kind`(시군구 전체, 최대 20개).
  - NEIS 학사일정 기반. 연속일 시험을 한 기간으로 병합하고 D-day로 정렬. 학사일정에 시험 미기재인 학교는 `_미기재_`로 명시(커버리지 한계 노출).
- **학교 비교 리포트** — 같은 시군구·학교급 학교를 핵심 지표로 한 표에 비교(전학·입학 결정용).
  - MCP `get_school_report`, CLI `schoolinfo report`, 웹앱 학교카드 `📋 학교 비교표`.
  - 지표: 학생수·학급당 인원(09)·급식 운영방식 직영/위탁(34)·**교원 기간제 비율**(22)·동아리 수(56). 시군구 전체를 항목당 1패스로 조회해 `SCHUL_CODE`로 조인.
  - `names`로 특정 학교만 추려 비교 가능.
- `LICENSE` 파일(MIT) 추가.

### Changed
- README: 신규 도구 2종·CLI 명령 2종·예시 반영. 제공 도구 수 로컬 12종·원격 11종으로 갱신.
- license 배지를 npm 조회형(`npm/l`) → 정적 MIT 배지로 교체. npm 미게시 시 `package not found`로 깨지던 문제 수정.

### Notes / Known limitations
- **급식비 집행(35)·학교도서관(58)** 공시는 OpenAPI가 `depthNo` 추가 필수 파라미터를 요구해(현재 미전달) 비교 리포트에서 제외했다. `depthNo` 값 확인 시 *1식 식품비·보호자 부담률·1인당 장서* 칼럼을 추가할 수 있다.
- 학교알리미 OpenAPI는 **최근 3년치만** 제공 → 장기 시계열(5년 추세 등) 미지원.
- 시험 캘린더의 지역 전체 모드는 학교 수만큼 NEIS를 조회하므로 상한 20개. 학사일정 미등록 학교는 집계에서 빠진다.
